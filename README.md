# 📊 Analiza Vânzărilor cu Python

## Descriere
Analiză completă a vânzărilor unui magazin de 
electronice folosind Python.

## Tehnologii folosite
- Python
- Pandas
- Matplotlib

## Ce face proiectul
- Citește date din CSV
- Calculează total și medie vânzări per produs
- Identifică cel mai bun produs și luna de vârf
- Generează grafice profesionale
